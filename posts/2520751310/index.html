简历

简历技能

1.熟悉 Kubernetes 集群搭建、配置与管理，实现 Master 节点高可用性；能够编写 YAML 文件，

实现 Pod 弹性伸缩及滚动更新。

2.掌握 Docker 架构与原理，能够编写 Dockerfile，构建、打包及推送镜像；了解 Docker Compose

使用，管理容器生命周期。

3.熟悉 Nginx 原理与功能，具备配置、优化及维护能力；实现反向代理、负载均衡等高级功能。

4.能够使用 Keepalived，实现 Nginx、Mysql、k8s 等高可用性。

5.能够部署与配置 Tomcat，实现性能优化，如内存分配、线程池等；搭建 Tomcat 集群。

6.熟悉 Ansible 配置，了解常用模块及使用方法，实现自动化运维。

7.掌握 Prometheus 监控体系与 Grafana 可视化工具，实现复杂的监控数据分析和展示。

8.熟练安装、配置及优化 Zabbix，自定义监控模版和图形，了解监控、图形化及告警机制。

9.了解 Jenkins 集成 Docker、k8s 等自动化工具，实现应用的自动化部署和扩展。

10.能够安装、配置及优化 Mysql，了解主从复制及 mgr 集群搭建。

11.了解 Redis 集群实现原理及部署方式，如 Cluster 集群，实现数据的水平扩展和容错能力。

12.能够部署与配置 Harbor 仓库，集成 Docker、k8s 等容器化平台，实现自动拉取和部署。

13.掌握 shell 脚本编写，实现自动化运维与监控，提高运维效率。

14.熟练掌握 Linux 基础命令与网络配置，熟悉系统监控工具，如 top、netstat 等。

15.了解 Java 基础与 SpringBoot 框架，能够使用 Eclipse、IDEA 等开发工具。
