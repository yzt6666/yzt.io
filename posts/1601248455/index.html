反爬虫之随机用户代理

发表时间：2023-05-16 阅读： 评论：

用一个用户代理一直爬取一个网页肯定会容易被检测到，所以需要我们每次要用不同的User-Agent来访问网页，访问时间间隔也要随机选取，才能使爬虫更像人的操作。

编写useragent.py文件

里面需要存储大量的User-Agent，存在一个列表中，越多越好。

user_list = [
    'Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/535.1 (KHTML, like Gecko) Chrome/14.0.835.163 Safari/535.1',
    'Mozilla/5.0 (Windows NT 6.1; WOW64; rv:6.0) Gecko/20100101 Firefox/6.0',
    'Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/534.50 (KHTML, like Gecko) Version/5.1 Safari/534.50',
    'Opera/9.80 (Windows NT 6.1; U; zh-cn) Presto/2.9.168 Version/11.50',
    'Mozilla/4.0 (compatible; MSIE 7.0; Windows NT 6.1; WOW64; Trident/5.0; SLCC2; .NET CLR 2.0.50727; .NET CLR 3.5.30729; .NET CLR 3.0.30729; Media Center PC 6.0; InfoPath.3; .NET4.0C; .NET4.0E) QQBrowser/6.9.11079.201'
]

引用文件加入到项目中

然后import文件导入到项目中就可以使用了，以下是用随机用户代理爬取的百度贴吧静态页面的源码。

from urllib import request,parse #爬取页面用到的库
import time,random  #random()是使用随机数的标准库，time()是处理时间的标准库
from useragents import user_list  #导入的用户代理列表
class Tieba():
    def __init__(self):
        self.url = 'https://tieba.baidu.com/f?{}'
    # 1.获取页面
    def get_html(self, url):
        header = {
            'User-Agent': random.choice(user_list)}  #随机选取列表中的代理
        req = request.Request(url=url, headers=header)
        res = request.urlopen(req)
        html = res.read().decode('utf-8')
        return html

    # 2.解析
    def parse_html(self):
        pass

    # 3.保存
    def save_html(self, filename, html):
        with open(filename, 'w', encoding='utf-8') as f:
            f.write(str(html))

    # 4.入口程序
    def run(self):
        name = input('请输入贴吧名：')
        begin = int(input('请输入起始页：'))
        end = int(input('请输入终止页：'))
        for page in range(begin, end + 1):
            # url的拼接
            pn = (page - 1) * 50
            params = {
                'kw':name,
                'pn':str(pn)
            }
            params = parse.urlencode(params)
            # 调用类中变量拼接文件名filename
            filename = '{}-第{}页.html'.format(name, page)
            url = self.url.format(params, pn)
            html = self.get_html(url)
            # 保存页面
            self.save_html(filename, html)
            # 爬虫操作更像人，爬取每页的时间随机
            time.sleep(random.randint(1, 2))


if __name__ == '__main__':
    start = time.time()
    spider = Tieba()
    spider.run()
    end = time.time()
    print('执行时间%.2f' % (end - start))
